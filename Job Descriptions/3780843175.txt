<strong>Description<br/><br/></strong>Are you seeking a new and challenging position supporting a complex Program supporting mission critical systems? Well, look no further! This is an exciting time to contribute to a program with lasting impactful results. Leidos is currently looking to add a Software Engineer to a Maintenance and Development task. The SWE will translate system requirements into software requirements, perform software design, and implement software applications and features in an agile environment. The SWE will maintain and modify systems while adding new features and capabilities. The candidate must have strong C++ and Matlab skills. Day to day activities include working with domain Subject Matter Experts and Software Engineers to fully understand system requirements and transition Matlab code to C++. Signal analysis or signal processing knowledge is preferred. System Administration skillset are useful for the maintenance task.<br/><br/><strong>Primary Responsibilities<br/><br/></strong><ul><li> Analyze user requirements to derive software design and performance requirements</li><li> Debug existing software and correct defects</li><li> Design and code new software or modify existing software to add new features</li><li> Integrate existing software into new or modified systems or operating environments</li><li> Develop simple data queries for existing or proposed databases or data repositories</li><li> Write or review software and system documentation</li><li> Serve as team lead at the level appropriate to the software development process being used on any particular project</li><li> Develop or implement algorithms to meet or exceed system performance and functional standards</li><li> Develop software solutions by analyzing system performance standards, confer with users or system engineers; analyze systems flow, data usage and work processes; and investigate problem areas</li><li> Modify existing software to correct errors, to adapt to new hardware, or to improve its performance</li><li> Implement recommendations for improving documentation and software development process standards</li><li> Confer with system engineers and/or hardware engineers to derive software requirements and to obtain information on project limitations and capabilities, performance requirements and interfaces</li><li> Coordinate software system installation and monitor equipment functioning to ensure operational specifications are met<br/><br/></li></ul><strong>Basic Qualifications<br/><br/></strong>Fourteen (14) years' experience as a SWE in programs and contracts of similar scope, type, and complexity is required. Bachelor's degree in Computer Science or related discipline from an accredited college or university is required. Four (4) years of additional SWE experience on projects with similar software processes may be substituted for a bachelor's degree.<br/><br/><ul><li>C++ advance knowledge and experience</li><li>Matlab knowledge and experience</li><li>TS/SCI with Polygrap<br/><br/></li></ul><strong>Preferred Qualifications<br/><br/></strong><ul><li>Signal processing knowledge/experience</li><li>Signal analysis knowledge/experience</li><li>System Administration, patching, updating systems in response to scans and vulnerabilities<br/><br/></li></ul><strong>Pay Range<br/><br/></strong>Pay Range $97,500.00 - $176,250.00<br/><br/>The Leidos pay range for this job level is a general guideline only and not a guarantee of compensation or salary. Additional factors considered in extending an offer include (but are not limited to) responsibilities of the job, education, experience, knowledge, skills, and abilities, as well as internal equity, alignment with market data, applicable bargaining agreement (if any), or other law.
