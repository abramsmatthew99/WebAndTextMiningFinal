DescriptionWe are currently seeking a Junior Software Engineer to support a high performing agile team using the Scaled Agile Framework (SAFe) methodology to support 10+ mission critical applications/systems on the Sustainment for Analysis Services (SAS) Program in Gaithersburg, MD. SAS provides system engineering, development, test, integration, and operational support to sustain applications/systems and perform adaptive maintenance infusing new technology and adding advanced capabilities while continuing to support the on-going mission of the customer. As a Junior Software Engineer, you will work on a specific tasking critical to SAS needs with opportunities to switch teams as you and our applications/systems grow and evolve. We need our engineers to be versatile, display leadership qualities and be enthusiastic to take on new problems as we continue to push technology forward.No clearance required to start. The candidate will be subject to a government security clearance and must meet eligibility requirements for access to classified information. The selected candidate will be nominated for required clearance upon hire.Primary ResponsibilitiesDesign, implement, test, deploy, maintain, and improve software in multiple software languages.Manage individual project priorities, deadlines, and deliverables.Use technical expertise to suggest environment and process upgrades.Realize system milestones and plan to deliver a quality product on time.Communicates with peers to explain facts, basic analysis, processes, and practices related to area.Basic QualificationsUS CITIZENSHIP IS REQUIRED PER CONTRACT.Bachelor's degree in Computer Science, Software Engineering, or Information Systems or equivalent experience and less than 2 years prior relevant experience.Knowledge of Java, JavaScript, web frameworks such as Node.js, Vue.js or React to build modern, dynamic user interfaces, and/or webservices.Experience with Virtual Machines.Experience with GitLab.Troubleshooting and problem-solving skills.Experience with Linux and Windows OS (scripting, command line).Proficiency in computer applications, languages, and technologies.Ability to obtain a TS/SCI clearance with Polygraph.Preferred QualificationsExperience with Agile software development cycle.Experience with Object-oriented programming languages to include C++, Python, and PHP.Experience with RESTful API using requests to access and use data.Experience with Spring Framework core features used by any Java application.Database experience with Oracle, SQL, and/or PostgreSQL.Java web developer with both server and client-side experienceExperience working in Cloud architecture and/or application transitions to the CloudFamiliarity with Cloud computing containers (i.e., Docker, Kubernetes, and Microsoft Azure)Experience with JIRA, and Confluence
