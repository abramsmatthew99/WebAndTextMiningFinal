<strong>Job Description<br/><br/></strong>Client seeks a Software Development Engineer in Test for the Bureau of Technology Development and Systems (BTDS). Reporting directly to the Deputy Director of BA, QA and UX, you will be responsible for creating test procedures for our Web &amp; Mobile Applications. You will understand Applications in-depth and design automated tests to validate their functionality &amp; performance. You will need a passion to find bugs and try to break an application.<br/><br/><strong>Responsibilities<br/><br/></strong><ul><li> Collaborate with product, engineering, and other domain teams to understand features, develop, and automate test cases, improve product quality, and shorten release time from dev to production.</li><li> Coordinating and driving E2E testing with cross-domains.</li><li> Being able to identify the manual processes and solve through automation.</li><li> Working with software engineers to understand product architecture and features.</li><li> Documenting and track bugs to closure.</li><li> Create and maintain data scenarios to support test plans and test cases.<br/><br/></li></ul><strong>Qualifications<br/><br/></strong><ul><li> Bachelor's Degree, Computer Science or similar, plus 8+ years of programming experience</li><li> Experience using automation tools such as Selenium, Appium, Cypress, Playwright or Microfocus QTP/UFT</li><li> Experience writing code using Java, JavaScript, and other similar languages including HTML &amp; CSS.</li><li> Experience with using defect management, test management tools, modern source control tools (Git/GitHub).<br/><br/></li></ul><strong>Preferred Skills<br/><br/></strong><ul><li> Experience in Continuous Integration and Continuous Deployment (CI/CD)</li><li> Experience creating strategies to cover vendor integrations.</li><li> Excellent SQL knowledge with the ability to write complex queries for validation as well as review stored procedures for verification.</li><li> Excellent analytical skills.</li></ul>
