<p><strong>Responsibilities:</strong></p><p>• Work closely with senior stakeholders to comprehend business needs and convert them into technical specifications.</p><p>• Create detailed technical plans for features or system design and documenting them clearly for development and implementation.</p><p>• Providing guidance and support to junior team members, helping them develop their skills, and promoting the use of industry best practices.</p><p>• Design, develop, and maintain SQL Server databases and applications using T-SQL.</p><p>• Write efficient, optimized, and complex T-SQL queries, stored procedures, functions, triggers, and views.</p><p>• Refactoring and optimizing T-SQL code for heterogeneous database migration.</p><p>• Run various tools and assess the report to compare the schema of source and destination databases.</p><p>• Help write and optimize in-application T-SQL statements.</p><p>• Perform data analysis, validation, and quality assurance.</p><p>• Troubleshoot and resolve database issues and performance problems.</p><p>• Migrate SQL Server databases to AWS using AWS Database Migration Service (DMS) and AWS Schema Conversion Tool (SCT) and Babelfish Compass tool.</p><p>• Work with AWS cloud services such as Amazon RDS, Amazon Aurora PostgreSQL, Amazon S3, Amazon EC2, and Amazon Lambda, Amazon Glue, SQL Server SSIS and DTS packages and XML code.</p><p>• Collaborate with cross-functional teams, developers, testers, and stakeholders on database development and migration projects.</p><p>• Document the processes and create runbooks for migration.</p><p> </p><p><strong>Requirements:</strong></p><p>• Bachelor’s degree in computer science, engineering, or related field.</p><p>• AWS Certified- Solution Architecture /+ any Data* stream.</p><p>• <strong>5-7+ years of experience as a T-SQL developer.</strong></p><p>• Expert in T-SQL programming and Microsoft SQL Server development tools.</p><p>• Experience with SQL Server performance tuning, indexing, and optimization.</p><p>• Experience with SQL Server Integration Services (SSIS) and SQL Server DTS Packages.</p><p>• Experience with AWS cloud services and database migration tools.</p><p>• Knowledge of AWS best practices and security standards.</p><p>• Understanding of code versioning tools such as GitHUB.</p><p>• Basic knowledge of PL/pgSQL is preferred.</p><p>• Excellent communication, problem-solving, and analytical skills.</p><p>• Ability to work independently and as part of a global team.</p><p></p>
