processing.</li><li> Designing, implementing, and testing real-time C/C++ software for multi-core systems running Linux or a real-time operating systems, such as VxWorks</li><li> Build software products that utilize third party mathematics and communication libraries</li><li> Interface communication standards / protocols such as TCP/IP, UDP, and DDS</li><li> Read and understand signal processing algorithms written in MATLAB</li><li> Work with Linux operating system as well as experience with real-time operating systems</li><li> Designing, implementing, and testing software to configure and control RF hardware devices</li><li> Integrating software into third-party software frameworks and architectures, such as Redhawk</li><li> Documenting the developed systems</li><li> Support system field testing and collection events, usually about a week in duration once or twice a year<br/><br/></li></ul><strong>The Successful Candidate Will Have The Following strong experience in C/C++ software development</li><li> Experience using version control (Subversion, GIT, etc.) and continuous integration software tools (Jenkins)</li><li> Must have active Secret level security clearance with the ability to obtain a TS/SCI.<br/><br/></li></ul><strong>Bonus Education/experience<br/><br/></strong><ul><li> Master’s Degree in Computer Engineering, Electrical libraries, such as TensorFlow</li><li> Experience with MATLAB GPU computing</li><li> Experience with High Performance Computing including parallel and distributed computation</li><li> Experience with Model-Based Design code generation practices</li><li> Experience with cloud-based computing platforms, such as Amazon Web Services (AWS)</li><li> Experience with other programming languages such as
