enhancements.</p><p><br/></p><p>• Identify and report software defects with detailed bug reports, including steps to reproduce.</p><p><br/></p><p>• Participate in code reviews to ensure automation code quality and best practices.</p><p><br/></p><p>• Continuously improve and optimize test scripts and frameworks for efficiency and maintainability.</p><p><br/></p><p>• Conduct performance testing automation engineers as needed.</p><p><br/></p><p><br/></p><p><br/></p><p>Qualifications:</p><p><br/></p><p>• Bachelor's degree in Computer Science, Software Engineering, or a related field (or or JUnit.</p><p><br/></p><p>• Hands-on experience with web automation tools like Selenium WebDriver.</p><p><br/></p><p>• Experience in API testing using tools like Postman, RestAssured, or similar.</p><p><br/></p><p>• Familiarity with version control systems (e.g., Git) and continuous integration/continuous deployment (CI/CD) pipelines.</p><p><br/></p><p>• Solid understanding of software testing concepts, methodologies, and best practices.</p><p><br/></p><p>• Strong analytical and problem-solving skills with meticulous attention to detail.</p><p><br/></p><p>• Excellent communication and collaboration skills.</p><p><br/></p><p>• Self-motivated with the ability to work independently and as part of a team.</p><p><br/></p><p>• Experience with Cypress for front-end testing is a plus.</p><p><br/></p><p>• Knowledge of SQL and database testing is a plus.</p><p><br/></p><p>• ISTQB or other relevant certifications are a plus.</p>
