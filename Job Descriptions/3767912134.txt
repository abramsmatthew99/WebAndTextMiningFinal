Write test strategy and test cases for the validation of various features and bug fixes on the DTEX software. Develop Test strategies and cases to cover positive and negative feature testing, A/B testing, performance testing and interoperability testing. Design and implement the automation of test cases. Participating in technical discussion and providing feedback from QA point of view. Understanding the existing automated test framework and enrich it as needed. Being able to create test automation from scratch. Requirements BS degree or equivalent in computer science or related field 3+ years of experience in Automation and Software Testing Excellent analytical and technical skills Strong knowledge of the Linux OS as an advanced System Administrator and User Skilled with automation with Selenium Ability to script in Python, Powershell, and Bash shell Good knowledge of virtualization tools like Virtual Box, VMWare ESXi and using them for automation Good understanding of underlying technologies used in Enterprise Software Applications such as Web Servers (Apache HTTPd) and Database Servers (PostgreSQL) Good understanding of networking concepts and protocols Good knowledge of security tools, and vulnerability scanners Good knowledge or Exposure to big data concepts and technologies like ElasticSearch, Hadoop, Redis and Apache Spark Experience with AWS A positive thinker, Risk taker, Going above and beyond the assigned tasks An outstanding team player Exact compensation may vary based on skills, experience, and location.
