        GAMA-1 seeks an experienced remote Application System Analyst to support the NESDIS Integrated Information Services-II (NIIS-II) contract. The Application System Analyst will be responsible for the administration, monitoring and maintenance of software infrastructures and applications creating tables, views, stored procedures, user defined functions, and SQL packages. He/she is considered an experienced “problem solver” and will primarily work with the SQL Server Database supporting the NOAA Research and Development (R&amp;D) Database (NRDD) which is a secure, web-based enterprise tool used to house project management data for all R&amp;D conducted by NOAA and its partnerships.<br/><br/>Duties/Responsibilities:<br/><br/>The Application System Analyst is tasked with overseeing practices and processes to ensure integrity, safety and availability of all data and applications as part of NESDIS’ information systems. He/she will ensure that the processes needed for users to function and succeed are running smoothly, acting as a technical point of contact to fix application and systems issues - usually on an immediate basis.<br/><br/>In this role, the Application System Analyst works alongside application developers and software engineers. Common job responsibilities will include:<br/><br/><ul><li>the training and support of staff in using applications</li><li>analysis and diagnosis of application errors</li><li>problem resolution in applications</li><li>identify and develop functionality changes</li><li>maintain documentation of key databases and linked applications</li><li>attend regular internal and external meetings to provide updates on the progress of your task and to present to users the application solutions you are proposing.<br/><br/></li></ul>Required Skills/Abilities:<br/><br/>The successful candidate must be
